# Vue + ES6 / Webpack / Express4

### Install instructions:
```sh
$ git clone https://github.com/allenRoyston/cake_test.git
$ cd cake_test
$ npm install
$ npm run dev
```

### Deployment instructions:
```sh
$ gulp build
$ git push heroku master
```

### Deployment preview (what you see is what you'll get):
```sh
$ gulp build
$ npm start
```
